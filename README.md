# FaceCard Scanner 💳

A fun web app that scans your face like a credit card and tells you if you "Slay" or get "Chopped"!

## Live Demo
[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://facecard-scanner.streamlit.app)

## Features
- 📸 Real-time face detection
- 💳 Credit card-style scanning interface
- ✨ Fun "Slay" or "Chopped" classification
- 🎨 Beautiful gradient UI with animations

## Deploy on Streamlit Cloud

1. Fork this repository
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub account
4. Select this repository
5. Deploy!

The app will automatically use:
- `requirements.txt` for Python dependencies
- `packages.txt` for system dependencies
- `app.py` as the main file

## Local Development
```bash
pip install -r requirements.txt
streamlit run app.py
